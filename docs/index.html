Demo: jquery-linear-navigation

Set focus on a list, then use ARROW keys. This plugin does not do any focus management, it only handles state, and triggers an event on state change.

This plugin consolidates common functionality and state of jquery-roving-tabindex and jquery-active-descendant. You probably don't need to use this plugin directly.

NOTE: This plugin does not require that keyboard focus go on the container of the items, it can go on any focusable element.
